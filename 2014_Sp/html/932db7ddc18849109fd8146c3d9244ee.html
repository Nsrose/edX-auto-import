Bytes

Bits can be represented in packages called bytes:

  • Byte = 8 bits
  • Kilobyte = 1024 or 2^10 bytes, 10^3 bytes
  • Megabyte = 10^6 bytes
  • Gigabyte = 10^9 bytes
  • Terabyte = 10^12
  • Petabyte = 10^15
  • Exabyte = 10^18
  • Zettabyte = 10^21
  • Yottabyte = 10^24
  • Hellabyte = 10^27

Hellabyte? Yes indeed. Google it.

How is data stored in bytes?

ASCII: How are letters represented by computers?

The letters of the alphabet have been mapped to specific binary numbers so they can be represented by computers. For example:
  • A = 65, or 0010 0001
  • B = 66, or 0010 0010
  • C = 67, or 0010 0011

So spelling CAB would be a total of 24 bits. You can look up the bits for the rest of the alphabet in an ASCII table. How many bits would a 750 word paper have? Assume you have an average of 5 letters per word and ignore spaces:

5 (letters per word) x 8 (bits per letter) x 750 (words) = 30,000 bits = 3,750 bytes = 3.7KB
