Experiment with Drawing Commands

Try to get comfortable with the pieces shown below. Figure out what each one does and try to use these pieces to draw a square or a simple picture.

Note
  • Once the pen is down, it stays down, even in a different script, until you say pen up.
  • It's easiest if your sprite wears the turtle costume (the arrowhead) so you can see the direction in which it's facing very clearly.
  • You also will want to show the direction and x and y position of the character. In the Motion tab you can select for these to be shown on the stage (like below).
