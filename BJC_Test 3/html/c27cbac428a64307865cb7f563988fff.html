Quadratic-Time Algorithms - O(n2)

A quadratic-time algorithm is one that takes a number of steps proportional to n2. That is, if the size of the input doubles, the number fo steps quadruples. A typical pattern of quadratic-time algorithms is performing a linear-time operation on each item of the input (n steps per item * n items = n2 steps).

Here are some examples:

  • Compare each item of a list against all the other items in the list
  • Fill in a n-by-n game board
