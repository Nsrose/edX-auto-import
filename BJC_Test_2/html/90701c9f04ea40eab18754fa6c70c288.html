Homework 3: Evil Hangman

Collaboration Policy: This assignment must be completed in groups of two.  You may discuss high-level ideas with other groups, but you should not share code.

Description

You unveiled your classic hangman game and discovered that your classic hangman game was too easy.  The other kids are kicking your computer’s butt left and right.  You decide to team up with another student to build a new, much harder hangman game, codenamed Evil Hangman.  

Evil Hangman is played in exactly the same way as the original game.  The dialog between the player and computer should look exactly the same as it did in your homework 2 solution.  In fact, the player should have no idea that s/he is not playing the original game and that the computer is actually cheating to make the game as hard as possible for the player to win.  Instead of just randomly choosing a secret word and sticking with it, the computer changes the secret word as the player starts guessing letters.

Evil Algorithm

The way the computer chooses the secret word is, of course, an algorithm albeit an evil one. First the user selects a word length. The computer will only keep all of the words from the list of that length. Then the user will select a letter and the computer will only keep all of the words that do NOT contain that letter and deduct a life. Continue until every word in the reduced list contains the user’s letter.

Now you have come to the point where you must reveal the positions where the user’s letters appear, but there are still multiple words that could be the secret word. Here we have to select the letter pattern to unveil.   A letter pattern refers the positions that will be revealed in a word if a particular letter is guessed. For instance, the letter pattern of the letter “L” in the word ball is  _ _ l l. It would be good to keep all of the words with the most common letter pattern in the list.  Probability-wise if you just pick a random word within the reduced list and use its letter pattern, you are very likely to pick the most common pattern. That’s good enough for us. Only keep the words from the reduced list that match letter pattern and reveal the letter pattern to the user. Remember to merge the previously reveal pattern with the new pattern each time. Continue until you only have one possibility for the secret word and then the game plays like classic hangman. Remember that, when played well, it should be possible to win a game of Evil Hangman.

Generalization

In the Evil Algorithm portion, we talked about a couple of ideas that would be great as custom blocks:

  • get pattern (letter) in (word): returns the letter pattern given a letter and word.

  • merge (pattern1) and (pattern2): merges a new pattern with the old pattern.
