Blocks using if and if-else

Let's try making a few more blocks that use if and if-else.

  • Make a block that when given a light color, makes the sprite say the appropriate action. For example should say "go!"
  • Make a block that takes in a percentage and makes the sprite say the associated letter grade. For example, should say "C."
  • Make a block that takes in a temperature and says whether water will be liquid, solid, or gas.
